# SPLIT-KEYBOARD-V2
The second attempt of creating the prefect split keyboard by @NotALarp
## WHAT IS IT?
This is my custom split keyboard made in KiCad, based on a LILY58 split keyboard design, it is sleek and is designed with ergonomics in mind. 

## Featured Hardware:
- Nice!Nano v2 microcontroller
- Lipo batteries
- Low profile choc switches
- 0.91" OLED displays
- EC11 Rotary Encoders

## WHY I MADE IT
My first split keyboard was pretty bad, it was based on a xiao nrf58240 microcontroller, had a terrible layout and overall wasn't that good. So I decided that with my previous knowledge that I would create a new, second split keyboard. 

## NOTES AND DESIGN CHOICES
This  time I decided to base the layout on a real split keyboard and some better microcontrollers. I decided on the nice!nano v2 and based the layout on a Lily58 split keyboard which I think looks really good. I designed a custom, 3d-printed case which was created to be compact, modern and sleek.
