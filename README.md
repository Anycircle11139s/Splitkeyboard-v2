I created a second split keyboard. My first split keyboard was pretty bad, it was based on a xiao nrf58240 microcontroller, had a terrible layout and overall wasn't that good. So I decided that with my previous knowledge that I would create a new, second split keyboard. This  time I decided to base the layout on a real split keyboard and some better microcontrollers. I decided on the nice!nano v2 and based the layout on a Lily58 split keyboard which I think looks really good. I designed a custom, 3d-printed case which was created to be compact, modern and sleek. I am really proud of what I created and can't wait to build it IRL.
